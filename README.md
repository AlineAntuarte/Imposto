# 🏛️ Calculadora de Imposto de Renda (Simulador IRRF)

Este projeto é uma ferramenta web robusta desenvolvida como parte da avaliação de **Frontend I** no curso de Análise e Desenvolvimento de Sistemas (IFSC). A aplicação foca na educação financeira e transparência tributária, permitindo entender o funcionamento do sistema de impostos brasileiro.

## 🚀 Requisitos da Avaliação Ganhos

Este projeto cumpre integralmente os seguintes requisitos acadêmicos:

- **Layout Fluido e Responsivo:** Adaptado para três tamanhos de tela específicos: Desktop, Tablet (720px) e Celular (540px).
- **Arquitetura de Grid:** Uso intensivo de **CSS Grid** adaptável para seções e formulários.
- **Efeito Parallax:** Implementação de fundo fixo com película de transparência na seção da Calculadora.
- **Performance de Imagens:** Utilização de três resoluções de imagens diferentes para cada breakpoint de tela, otimizando o consumo de dados.
- **Navegação Avançada:** Header fixo, botões de avanço de seção e link fixo para retorno ao topo.

## 🛠️ Tecnologias e Padrões

- **HTML5 Semântico:** Uso de tags como `<header>`, `<nav>`, `<section>`, `<footer>` e elementos interativos nativos como `<details>` e `<summary>`.
- **CSS3 Moderno:** Aplicação de Variáveis CSS (`:root`), Flexbox, CSS Grid e seletores avançados como `:first-of-type`.
- **Validação W3C:** Código 100% aprovado pelos validadores oficiais do W3C para HTML e CSS.
- **Acessibilidade:** Estrutura de cabeçalhos (H1-H4) organizada logicamente para leitores de tela.

## 📋 Funcionalidades Implementadas

- [x] **Glossário Interativo:** Organizado com efeito sanfona (accordion) nativo em HTML5 para melhor UX.
- [x] **Formulário de Contato:** Organizado via Grid com campos de Nome, Sobrenome, País e Mensagem.
- [x] **Educação Tributária:** Seção completa explicando a função e a redistribuição do imposto de renda.
- [x] **Navegação Inteligente:** Scroll suave e botões de navegação rápida entre seções principais.
- [ ] **Lógica de Cálculo (JS):** Cálculo automático de INSS e IRRF (Próxima etapa de desenvolvimento).
